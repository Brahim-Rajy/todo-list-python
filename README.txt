Project 1: A basic To Do List program
The user has 4 choice:
1 - add a task
2 - edit task
3 - delete a task
4 - showing all tasks
5 - An idea for timer, you just enter task name and how much times you want to be in this task in minutes
  - Then the timer start counting until 00:00
6 - exit program with option if you want to sauvegarde your data in a texte file(for human reading) and JSON file(for program reading)
